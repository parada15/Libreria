Botones

Los botones son elementos que se utilizan para realizar acciones en una página web. Los botones pueden ser de diferentes tipos, tamaños y colores. Puedes utilizar estos estilos de botones para tus proyectos web.

<button class="btn-cuadrado btn-grande fondo-blanco color-azul text-lowercase">cuadrado</button>
<button class="btn-redondo btn-grande">redondo</button>
<button class="btn-sredondo btn-peque color-magenta">sredondo</button>
<button class="btn">por_defecto</button>

Tipo de Texto

Para el diseño del texto contamos con la clase .text que es una clase
que se puede utilizar en elementos de texto .
esta cuenta con algunas subclase
.text-uppercase: se utiliza para convertir todo el texto de un elemento en mayúsculas.
.text-lowercase: se utiliza para convertir todo el texto de un elemento en minusculas.
.text-bold: se utiliza para hacer el texto de un elemento más grueso y oscuro, para enfatizarlo.
.text-italic: se utiliza para hacer el texto de un elemento cursivo, lo que puede ser útil para indicar énfasis o para citar palabras de otro idioma.

Mover elementos

Para este diseño contamos con la clase .flexbox que es una clase
que se puede utilizar para mover elemtos .
esta cuenta con algunas subclase
Puedes utilizar la clase .flexbox y sus diferentes modificadores (por ejemplo, .flexbox--row) que se pueden aplicar para ajustar la dirección de flujo del contenido, el espaciado y la alineación de los elementos. Asi mismo los valores:flex, flex-direction, flex-wrap, justify-content, align-items, align-content, flex-grow, flex-shrink y flex-basis.

Tambien puedes utilizar la clase .grid y sus diferentes modificadores (por ejemplo, .grid--row-dense) que se pueden aplicar para ajustar la dirección y el flujo de la cuadrícula, el espaciado entre los elementos y la asignación automática de espacio. Asi mismo los valores: grid, grid-template-columns, grid-template-rows, grid-gap, grid-auto-flow y grid-auto-columns.

Ocultar elementos

Solo necesitas el siguiente codigo en html agregado a una clase cualquiera que desees ocultar,hace referencia a la clase utilitaria "ocultar-ocul" y "ocultar-hi"

ocultar-ocul: oculta el elemento y su espacio ocupado

ocultar-hi: Oculta el elemento dejando siempre el espacio

ocultar-ocul ocultar-hi

Colores de fondo

fondo-[rojo, Azul,celeste,gris,verde,negro,chocolate,rojo,amarillo,blanco,rosa,degrada]
